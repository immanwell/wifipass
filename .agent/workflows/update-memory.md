---
name: Update Memory Bank
command: /update-memory
---
**GOAL:** Persist changes and progress to the Memory Bank.
1.  ANALYZE the entire session history since the last `/load-context`.
2.  PROPOSE a change to `memory-bank/activeContext.md` summarizing accomplishments and defining the next task.
3.  PROPOSE updates to the checklist in `memory-bank/progress.md`.
4.  ASK the user: "Review the proposed updates. Confirm to save, or provide edits."
5.  **EXECUTE**: If confirmed, write the changes to the respective files.
