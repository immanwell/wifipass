A secure, modern, and beautiful way to share Wi-Fi access via QR codes. Built for simplicity and visual appeal.

![WifiPass Screenshot](https://raw.githubusercontent.com/immanwell/wifipass/main/public/app-screenshot.png)

## Features

*   **Beautiful UI**: Glassmorphism design with a "Midnight & Neon" aesthetic.
*   **Secure**: Generates standard WPA/WPA2/WEP QR codes.
*   **Print Ready**: Layout automatically adjusts for A4 printing to create physical guest cards.
*   **Privacy First**: No data is stored; everything is generated on the fly.
*   **Responsive**: Works flawlessly on desktop and mobile.

## Tech Stack

*   **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS
*   **Backend**: Node.js, Express, TypeScript
*   **Design**: Custom CSS for glassmorphism and print layouts

## Quick Start

1.  Clone the repo:
    ```bash
    git clone https://github.com/immanwell/wifipass.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the dev server:
    ```bash
    npm start
    ```
4.  Open `http://localhost:3000`

## Deployment

Designed to be self-hosted (e.g., Coolify, Docker, or CloudPanel).
